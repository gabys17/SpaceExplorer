Instructions

Objective:

Avoid 5 shoots from the alien and get to the portal to be back safe at home.

Controls:

  • Arrow Left key to move left
  • Arrow Right key to move right
  • Arrow Up key to jump

Gameplay:

  • You start the game with five lives.
  • Avoid being shoot by the alien, as each shoot will cost you one life.
  • If you lose all five lives, the game is over.
  • The game has a 60-second timer.
  • If you reach the portal you win.
